# Design references

Reference material gathered while building the ASG UI surfaces. This is not
application code; nothing here is imported by the backend or the dashboards.

- `references/` — design-language extractions and previews for sites used as
  visual references (e.g. alexstoykovgroup.com, apple.com).
- `extract-output/` — output of the design-extraction tooling (style tokens,
  variables, screenshots).
- `idx/` — IDX/MLS styling references (Squarespace global CSS, category styles).

These are kept in the repo for provenance and to document the visual decisions
behind the live surfaces in `apps/admin-hub`.
